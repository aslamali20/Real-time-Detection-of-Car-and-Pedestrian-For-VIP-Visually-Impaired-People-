# Real-time-Detection-of-Car-and-Pedestrian-For-VIP-Visually-Impaired-People-
A computer vision-based model to detect two objects, person and car, and raise a voice alert. In this work, ORB feature descriptor is used. The feature vector dimensionality is reduced by using the elbow method to decide the count of K-means clusters. For faster object detection with reduced hardware we used PCA.
